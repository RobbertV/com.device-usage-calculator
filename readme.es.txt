Usage Mate está ahí para calcular y dar información sobre el uso y los costos de cualquier dispositivo.
Está optimizado para trabajar con precios de energía dinámicos o cambiantes según la hora del día.

1. Crea un dispositivo virtual Usage Mate
2. Elige el tipo de medición que deseas (kWh, W, m3, GJ, l, g, oz, gal, ft3)
3. En la configuración del dispositivo, puedes opcionalmente establecer la moneda preferida
4. Usa las tarjetas de acción del dispositivo para iniciar, actualizar y detener una sesión de uso
5. El resultado de la sesión se almacenará en el dispositivo y se puede usar dentro de los flujos para informar los resultados
