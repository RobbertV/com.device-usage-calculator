Usage Mate er der for å beregne og gi innsikt i bruk og kostnader for enhver enhet.
Det er optimalisert for å fungere med dynamiske eller skiftende energipriser basert på tid på dagen.

1. Opprett en Usage Mate virtuell enhet
2. Velg hvilken type måling du ønsker (kWh, W, m3, GJ, l, g, oz, gal, ft3)
3. I enhetens innstillinger kan du eventuelt angi ønsket valuta
4. Bruk enhetens handlingskort for å starte, oppdatere og stoppe en bruksøkt
5. Resultatet av økten vil bli lagret i enheten og kan brukes i flyter for å rapportere resultatene
