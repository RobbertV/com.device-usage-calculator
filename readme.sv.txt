Usage Mate är där för att beräkna och ge insikter om användning och kostnader för alla enheter.
Det är optimerat för att fungera med dynamiska eller växlande energipriser baserat på tid på dagen.

1. Skapa en Usage Mate virtuell enhet
2. Välj vilken typ av mätning du vill ha (kWh, W, m3, GJ, l, g, oz, gal, ft3)
3. I enhetens inställningar kan du valfritt ställa in önskad valuta
4. Använd enhetens åtgärdskort för att starta, uppdatera och stoppa en användningssession
5. Resultatet av sessionen kommer att lagras i enheten och kan användas i flöden för att rapportera resultaten
