Usage Mate предназначен для расчета и предоставления информации об использовании и затратах на любое устройство.
Он оптимизирован для работы с динамическими или изменяющимися ценами на энергию в зависимости от времени суток.

1. Создайте виртуальное устройство Usage Mate
2. Выберите тип измерения, который вы хотите (кВтч, Вт, м3, ГДж, л, г, унция, галлон, фут3)
3. В настройках устройства вы можете при необходимости установить предпочитаемую валюту
4. Используйте карты действий устройства для начала, обновления и остановки сеанса использования
5. Результат сеанса будет сохранен в устройстве и может быть использован в потоках для отчета о результатах
